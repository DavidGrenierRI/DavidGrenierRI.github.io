Real Jobs RI Report

Prototypes

The Real Jobs Report prototypes have been updated with the last round of feedback.

We've adjusted the Performance section as requested. Please review to confirm we understood the requests properly. The three large blocks for Placement Rate, Average Wage after Placement and Average Wage Gain represent a greater information weight than the other indicators. They may not be laid out as three large blocks but they should be styled to be much more prominent and imediately draw the viewers eye.

We've added a color change on hover for the clickable tiles. This will indicate to the designer that there should be something that happens there to make it clear to a user that a tile is clickable.

We've mocked up views for the Sector Partnerships and Employers Served details. We used the table of data on sector partnerships from the current RJRI site for content. We made up dummy content for the Employers Served view. There are four options for how detail views are displayed:

  • Version A - The user clicks on a tile and is taken to a detail page for that indicator.
  • Version B - The detail view is presented in a lightbox.
  • Version C - A summary view is presented in a lightbox, with the option to click through to a details page.
  • Version D - The detail view is presented on a separate page, but the indiator definition is included in the links title attribute so it will display on hover in most (non touchscreen) browsers.

Of these four options I would recommend Version A.
